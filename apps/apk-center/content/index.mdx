# AZIT APK Center에 오신 것을 환영합니다

AZIT 안드로이드 앱의 최신 APK 파일을 여기서 다운로드할 수 있습니다.

## APK 다운로드

[APK 다운로드 페이지](/downloads)에서 최신 버전을 다운로드하세요.

## 설치 방법

1. [APK 다운로드](/downloads) 페이지에서 원하는 버전을 다운로드합니다.
2. 안드로이드 기기의 **설정 > 보안** 메뉴에서 **알 수 없는 출처** 설치를 허용합니다.
3. 다운로드된 APK 파일을 열어 설치를 진행합니다.
4. 설치 완료 후 앱을 실행합니다.

> **참고:** Android 8.0 이상에서는 설치 시 해당 앱(파일 관리자, 브라우저 등)에 대해 개별적으로 설치 권한을 허용해야 합니다.

## APK 추가 방법 (관리자용)

새로운 APK를 배포하려면:

1. 빌드된 `.apk` 파일을 `static/downloads/` 폴더에 복사합니다.
2. `pnpm build` (또는 `gatsby build`)를 실행하여 사이트를 재빌드합니다.
3. 빌드 결과물을 배포하면 [다운로드 페이지](/downloads)에 자동으로 표시됩니다.
